Easily create apps using the web technologies you know and love: HTML, CSS, and JavaScript

I am a free and open source framework that allows you to create mobile apps using standardized web APIs for the platforms you care about.

To put it simply, I take the pain out of developing mobile apps

What makes me great is my ability to access native hardware and software features of many devices

  • Accelerometer
  • Tap into the device’s motion senstor. Capture device motion in the x, y, and z direction.
  • Contacts
  • The contacts object provides access to the device contacts database.
  • Media
  • The Media object provides the ability to record and play back audio files on a device.
  • Geolocation
  • The geolocation object provides access to the device's GPS sensor.
  • Capture
  • Provides access to the audio, image, and video capture capabilities of the device, including camera and microphone.
